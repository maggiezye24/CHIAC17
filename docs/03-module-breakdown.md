# Module Breakdown

## Core Modules (Minimum Requirements)

### Authentication Module
- User registration and login
- Secure password storage
- Role-based access control

### Quiz Management Module
- Quiz creation and editing
- Question management
- Quiz publishing control

### Quiz Attempt Module
- Start quiz attempts
- Capture user responses
- Automatic scoring
- Attempt storage

### Attempt History Module
- View previous attempts
- Access detailed attempt results

## Extended Modules (Enhancements)

### Analytics Module
- Learner performance summary
- Score trends
- Topic and difficulty analysis

### Integrity Monitoring Module
- Track suspicious behaviors
- Generate integrity score

### Recommendation Module
- Quiz recommendations
- Skill level estimation (optional)
